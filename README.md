# Cuore Italiano Landing

## Run
```bash
npm install
npm run dev
```

## Build / Vercel
```bash
npm run build
```
Deploy with Vercel default Vite settings.

## Asset assumptions
- `/public/assets/cuore-logo.svg` should be replaced with the official attached logo file if a production raster/vector original is available.
- Product imagery uses Cuore Italiano website upload URLs as the visual base and includes a branded fallback asset if a remote image URL changes.
